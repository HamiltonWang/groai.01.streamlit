# GroAI.01 Streamlit Dashboard

A professional AI Signal Mapping Dashboard for **AIHedge.finance**. This application interfaces with the GroAI API to simulate LLM outputs, visualize expected returns, and analyze market signals dynamically.

## üöÄ Overview

The **GroAI.01 Dashboard** serves as a frontend interface for the AI forecasting engine. It allows analysts and traders to:
- **Input & Simulate**: Provide custom prompts and modify feature vectors (history and technicals).
- **Visualize Returns**: Compare expected returns generated by different models (LLM-simulated, Historical Statistics, and ML Models).
- **Dynamic Interaction**: Automatically syncs feature definitions with the backend API.

## üñ•Ô∏è Features

- **Sidebar Controls**:
    - **Prompt Input**: Define the context for the signal (e.g., "bullish momentum").
    - **History Features**: Adjust normalized values for historical feature sets.
    - **Model Technicals**: Input specific technical indicator values (EMA, RSI, MACD).

- **Analysis Results**:
    - **LLM Output**: Structured JSON output simulating the language model's reasoning.
    - **Return Metrics**: Side-by-side comparison of `LLM`, `History`, and `Model` expected returns.
    - **Charts**: Visual bar chart comparison for quick decision support.
